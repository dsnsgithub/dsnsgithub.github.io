Creating a Minecraft Server

Creating a Minecraft server is easier then you think. We will need a few requirements before we start.

Requirements

1. Java

        Java is the first requirement you will need. 

        If you are on Windows or Mac, download Java here: https://www.java.com/en/download/


        If you are Linux, grab Java from your repo of choice.

2. Ability to port forward

        If you are only making a Minecraft server to use locally, also known as LAN, then you will not need this requirement.
        If you are planning to make this public to people outside your network, you will need to port forward.


        You need to know the IP Address to your default gateway, which you can find by typing
ipconfig

for Windows, or

ifconfig 
        for Linux users.

        You will also need to know the password to your router/default gateway. 
        Unless you know someone has changed it, it is usually the default router password, which you can find online.


        Inside the router settings, there is usually a setting to port forward.

        Make sure that the port is 25565, and the local IP address is whatever is displayed in ifconfig/ipconfig.

Making the Minecraft server

Step 1: Downloading the server.jar

        We will be using paper as our server, since it has the most bug fixes and is easy to install.

        Download the latest version of paper at https://papermc.io/downloads
        and rename the jar to
paper.jar

Step 2: Running the Jar

        This will be the final step. Running the jar is suprisingly easy. 

        Move the jar into an empty folder, and open your command line. 
        Move to the directory, and type
java -Xms3G -Xmx3G -jar paper.jar nogui 

        and replacing 3GB of ram with whatever amount you need.

        Do NOT allocate more ram then half of your system memory.

        After you started up the server, you are done! Enjoy playing on your Minecraft server!

        If you want better performance, use these flags

java -Xms3G -Xmx3G -XX:+UseG1GC -XX:+ParallelRefProcEnabled -XX:MaxGCPauseMillis=200 -XX:+UnlockExperimentalVMOptions -XX:+DisableExplicitGC -XX:+AlwaysPreTouch -XX:G1NewSizePercent=30 -XX:G1MaxNewSizePercent=40 -XX:G1HeapRegionSize=8M -XX:G1ReservePercent=20 -XX:G1HeapWastePercent=5 -XX:G1MixedGCCountTarget=4 -XX:InitiatingHeapOccupancyPercent=15 -XX:G1MixedGCLiveThresholdPercent=90 -XX:G1RSetUpdatingPauseTimePercent=5 -XX:SurvivorRatio=32 -XX:+PerfDisableSharedMem -XX:MaxTenuringThreshold=1 -Dusing.aikars.flags=https://mcflags.emc.gs -Daikars.new.flags=true -jar paper.jar nogui

More information about these flags can be found here.
